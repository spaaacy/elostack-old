# EloStack (Old)

[Demo Video](https://www.youtube.com/watch?v=cotuNW5tZTY)

This repo contains the old code for the first 3 versions of EloStack before shifting to a new repo with a fresh codebase. Built using Next.js, React, JavaScript, TailwindCSS, Stripe, CRON, Supabase, and PostgreSQL, and deployed on Vercel.
